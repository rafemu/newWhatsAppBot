# אימות והרשאות - Epic

## דרישות טכניות
- JWT לאימות
- bcrypt להצפנת סיסמאות
- MongoDB לאחסון נתונים
- Express middleware לאבטחה
- Validation schemas עם Joi/Yup

## מדדי הצלחה
- זמן תגובה מהיר (< 300ms)
- אבטחה גבוהה
- כיסוי בדיקות > 90%
- תמיכה בריבוי משתמשים במקביל
